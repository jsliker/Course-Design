### 初识git

#### 介绍

GIT 是一个分布式版本控制软件，最初由林纳斯·托瓦兹（Linus Torvalds）创作，于2005年以GPL发布。最初目的是为更好地管理Linux内核开发而设计。是目前世界上最先进的分布式版本控制系统.

#### git的安装

使用下面的命令在Linux系统下安装git工具.
```
$ sudo apt-get install git
```
